Аудиоредактор

Автор: Смирнов Иван

Описание:
Данное приложение предназначается для редактирования аудио.
Поддерживаются следующие возможности:

1. Вставка/удаление фрагментов
Использование: 
Ctrl+C, Ctrl+V
Чтобы выделить весь фрагмент трека нужно нажать Ctrl+A
Чтобы перейти в конец трека нужно нажать Ctrl + ->
Аналогично, для перехода в начало Ctrl + <-

2. Затухание/появление из тишины.
Использование:
На правой панели трека Effects -> Fade out / Fade in

3. Регулировка громкости на участке трека.
Использование: 
На правой панели трека Adjust loudness
Также если выделена часть трека - громкость меняется только для выделенной части

4. Переворот участка трека.
Использование: 
Выделение части трека, затем комбинация клавиш Ctrl+R развернёт его

5. Сложение нескольких треков в один.
Использование:
У каждого трека есть кнопка on/off.
Их можно переключать нажатием между двумя положениями.
При нажатии на Alt+S производится суммирование всех треков, у которых установлен параметр on.

6. Изменение скорости и высоты звука.
Использование:

7. Компрессия звука (усреднение фрагментов).
Использование: 
На правой панели трека Effects -> Compress sound
Параметр ratio отвечает за силу компрессии.
Он должен быть от 0 до 1.
Чем больше значение параметра ratio - тем сильнее усредняется звук.
