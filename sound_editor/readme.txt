Аудиоредактор

Автор: Смирнов Иван

Описание:
Данное приложение предназначается для редактирования аудио.
Поддерживаются следующие возможности:
1. Вставка/удаление фрагментов
2. Затухание/появление из тишины.
3. Регулировка громкости на участке трека.
4. Переворот участка трека.
5. Сложение нескольких треков в один.

Требования
1. Python версии не ниже 3.5.1
2. PyQt версии 4, numpy.

Состав
1. Графическая версия main.py
2. Ядро для работы со звуком Core/... (можно использовать как библиотеку)
3. Тесты Tests/...

Подробности реализации описаны в файле decomposition.txt
